Política de Privacidad

Esta aplicación ("Simone Marketing") es un panel interno de gestión de comentarios y mensajes directos de Instagram para la cuenta de negocio asociada.

Datos que procesamos

A través de la API de Instagram, la aplicación accede a: comentarios en publicaciones, mensajes directos, e información básica del perfil de Instagram conectado (nombre de usuario, identificador). Estos datos se usan únicamente para mostrar y responder comentarios y mensajes desde el panel.

Almacenamiento

El token de acceso de Instagram y una caché temporal de comentarios/conversaciones se almacenan de forma cifrada en una base de datos privada (Upstash Redis), accesible solo por el operador de esta aplicación.

Compartición de datos

No compartimos, vendemos ni cedemos estos datos a terceros. Los datos no se usan con fines publicitarios.

Eliminación de datos

Para solicitar la eliminación de los datos almacenados asociados a tu cuenta de Instagram, contacta a: goyochurches@gmail.com.

Contacto

Para cualquier consulta sobre esta política, escribe a goyochurches@gmail.com.
